
# 🚀 ميثاق الإطلاق السيادي النهائي (king2026 Edition)

لقد تم تحصين الكوكب القانوني رقمياً. اتبع هذا البروتوكول لإعلان السيادة الآن:

## 1️⃣ الإطلاق عبر Vercel (المسار الموصى به)
1. سجل دخولك إلى [Vercel.com](https://vercel.com).
2. اضغط على **"Add New Project"**.
3. قم باستيراد مستودعك.
4. في قسم **Environment Variables**، أضف المفاتيح التالية (من ملف config.ts):
   - `NEXT_PUBLIC_FIREBASE_API_KEY`
   - `NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN`
   - `NEXT_PUBLIC_FIREBASE_PROJECT_ID`
   - `NEXT_PUBLIC_FIREBASE_APP_ID`
   - `GEMINI_API_KEY` (للمحرك الذكي)
   - `DATABASE_URL` (لرابط Neon)
5. اضغط **Deploy**.

## 2️⃣ الرابط الحي (Live Portal)
بمجرد اكتمال البناء، سيمنحك Vercel رابطاً مثل:
`https://almustasharai-ai.vercel.app`

## 🛡️ ملاحظات المالك king2026:
- **البوت الذكي**: جاهز لخدمة المواطنين برصيد ٥٠ EGP ترحيبي.
- **المتحكم السيادي**: متاح لك فقط في لوحة الإدارة لإصدار أوامر الحظر والشحن آلياً.
- **التحديث**: أي تعديل أفعله لك هنا، سيقوم Vercel بتحديثه في موقعك الحي **تلقائياً** بمجرد نسخه لـ GitHub.
