## 2. مصفوفة الوظائف (Functional Matrix)

### A. محرك "المستشار الذكي" (Citizen Core)
- **الصلاحية**: مفتوح لكافة المواطنين المسجلين.
- **القدرات**:
    - **Vision Protocol**: تحليل صور البطاقات، العقود، والمستندات عبر الكاميرا أو الرفع.
    - **Vocal Protocol**: إملاء الأوامر والأسئلة بالصوت (Speech-to-Text).
    - **Smart Sidebar**: قائمة تضم (صياغة عقود، تحليل مخاطر، شكاوى عمالية، قضايا أسرة).
- **التكلفة**: 1 وحدة سيادية لكل استشارة.

### B. غرفة العمليات "المستشار السيادي" (Supreme Hub)
- **الصلاحية**: حصرية للمالك king2026 فقط.
- **القدرات**:
    - **Autopilot Mode**: مراقبة المحادثات الحية وحظر الانتهاكات آلياً.
    - **Executive Commands**: أوامر شات مباشرة (حظر @مستخدم، شحن 100 @مستخدم، مسح سجلات النظام).
    - **Dashboard Sync**: عرض إحصائيات المواطنين والخبراء لحظياً.

### C. الخزنة السيادية (The Vault)
- **الوظيفة**: بوابة شحن الأرصدة وإدارة الباقات.
- **التصميم**: يحاكي خزائن البنوك السويسرية الرقمية.
- **المسار**: ربط آلي مع Vodafone Cash و Stripe.

## 3. المعايير التقنية (Technical Sovereignty)
- **Framework**: Next.js 15 (App Router).
- **State Management**: React Context + Firestore Real-time Snapshots.
- **Security Protocol**: 
    - Firebase Security Rules (Level 4 Protection).
    - Smart Shield: حظر آلي للمحتوى المسيء أو الكلمات المحظورة.
- **Database**: PostgreSQL (Neon) للمهام المعقدة، و Firestore للبيانات اللحظية.
