# 📜 التقرير الشامل لمشروع المستشار AI (Almustashar AI)

## 1. نظرة عامة على المشروع
**المستشار AI** هو منصة SaaS قانونية سيادية متكاملة، تهدف إلى رقمنة الاستشارات القانونية في المنطقة العربية باستخدام أحدث تقنيات الذكاء الاصطناعي. يوفر النظام بيئة آمنة، احترافية، وفائقة الرفاهية للمستخدمين والخبراء القانونيين.

---

## 2. الهيكل التقني (The Tech Stack)
تم بناء المنصة باستخدام بنية برمجية عالمية تضمن السرعة والأمان:
- **الإطار البرمجي (Frontend)**: [Next.js 15] مع [React 19] لضمان أداء فائق وتجاوب ذكي.
- **التصميم والواجهة (UI/UX)**: [Tailwind CSS] مع فلسفة **Cosmic Glassmorphism** (الزجاج الكوني) و [Framer Motion] للتحريك الاحترافي.
- **قاعدة البيانات والخدمات (Backend)**: [Firebase] (Firestore, Auth, App Hosting) لضمان التزامن اللحظي للبيانات.
- **الذكاء الاصطناعي (AI Brain)**: [Google Gemini 2.5 Flash] عبر إطار عمل [Genkit] لمعالجة الاستشارات القانونية الرصينة.

---

## 3. المميزات الوظيفية الكبرى

### 🤖 مركز قيادة البوت (The Bot Hub)
- واجهة مستوحاة من **ChatGPT** و **Replit**.
- شريط جانبي (Sidebar) ذكي لإدارة سجل المحادثات.
- **وسائل راحة فائقة**:
    - ميكروفون للإملاء الصوتي (Speech-to-Text).
    - كاميرا لالتقاط المستندات وتحليلها فورياً.
    - نظام إرفاق ملفات (PDF, Word).
- أزرار أوامر سريعة لبدء المهام القانونية بضغطة زر.

### 📄 المكتبة السيادية (The Document Mint)
- أرشيف يضم مئات النماذج والعقود المعتمدة.
- نظام تخصيص الوثائق ببيانات الهوية الوطنية.
- إصدار فوري لوثائق **PDF** معتمدة ومشفرة.

### 👨‍💼 غرفة القيادة العليا (Supreme Admin Hub)
- تحكم مطلق للمالك في:
    - **إدارة المواطنين**: حظر، فك حظر، أو حذف المستخدمين.
    - **إدارة الخبراء**: تسجيل وحذف المستشارين المعتمدين.
    - **محرك الرقابة**: التحكم في قائمة الكلمات المحظورة.
