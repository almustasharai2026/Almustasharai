
# 🏛️ ميثاق التأسيس السيادي: المستشار AI (إصدار الذكاء المزدوج)

هذا المستند هو المرجع التقني والوظيفي النهائي لمنصة **المستشار AI**. تم تصميمه ليكون بروتوكول بناء لا يقبل الخطأ، مع دمج عقلية الذكاء الاصطناعي في كل خلية من خلايا النظام.

## 1. الهوية البصرية السيادية (Visual DNA)
- **فلسفة التصميم**: Ultra-Glassmorphism (تأثير الزجاج البلوري مع الظلال العميقة).
- **لوحة الألوان**:
    - الأساسي: **Midnight Cosmic Blue** (#020617).
    - التميز: **Sovereign Gold** (#D4AF37).
    - التفاعل: **Electric Indigo** (#6366f1).
- **الخطوط المعتمدة**: Cairo (للنصوص والبيانات) و Tajawal (للعناوين والمواثيق الملكية).

## 2. الهيكل البرمجي المزدوج (Hybrid Architecture)
- **Frontend**: Next.js 15 (App Router) مع React 19 لضمان أداء فائق السرعة واستجابة لحظية.
- **Backend-as-a-Service**: Firebase لعمليات التزامن والدردشة والرقابة اللحظية.
- **Transactional DB**: PostgreSQL (Neon) مع Drizzle ORM للعمليات المالية والإدارية المعقدة.
- **AI Core**: Genkit + Gemini 2.5 Flash للتحليل القانوني الاستراتيجي.

## 3. مصفوفة المحركات الذكية (Intelligence Matrix)

### A. محرك "المستشار الذكي" (Citizen Core)
- **المهمة**: خدمة المواطنين وتوفير وسائل الراحة الفائقة.
- **القدرات**:
    - **Vision Protocol**: معالجة صور المستندات والوثائق عبر الكاميرا السيادية.
    - **Speech-to-Text**: دعم الإملاء الصوتي للأوامر والاستشارات.
    - **Quick Actions**: نظام بطاقات ذكي لتوليد (عقود، خطابات، تقارير) بضغطة زر.
- **التكلفة**: ١ وحدة سيادية (EGP) لكل رسالة.

### B. محرك "المستشار السيادي" (Supreme Autopilot)
- **المهمة**: المساعد الشخصي والمتحكم الآلي للمالك king2026.
- **المكان**: حصري في غرفة القيادة العليا (Admin Page).
- **القدرات**:
    - **Command Executive**: تنفيذ أوامر الشات المباشرة (حظر @مستخدم، شحن ٥٠٠٠ @مستخدم).
    - **Autonomous Monitoring**: مراقبة الكوكب لحظياً وإصدار تنبيهات عند رصد انتهاكات.
    - **Sovereign Reporting**: توليد تقارير إحصائية وتقنية شاملة بطلب واحد.

## 4. نظام السيادة المالية (Economic Protocol)
- **الرصيد الترحيبي**: 50 EGP لكل مواطن جديد يتم توثيق هويته.
- **الخزنة (The Vault)**: واجهة شحن أرصدة زجاجية تدعم فودافون كاش و Stripe.
- **الشفافية**: سجل كامل لكل وحدة مالية يتم خصمها أو إضافتها.

## 5. الأمان والرقابة (Smart Shield)
- **الدرع الواقي**: نظام فحص لحظي ينهي الجلسات ويحظر المستخدمين آلياً عند رصد كلمات محظورة.
- **التوثيق المهني**: معالج تصوير (Wizard) لتوثيق كارنيهات النقابة وبطاقات الهوية للمحامين.
