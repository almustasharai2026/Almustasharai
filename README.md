# 🏛️ المستشار AI - كوكب العدالة الرقمية

مرحباً بك في المنصة القانونية الأكثر تطوراً في المنطقة العربية. **المستشار AI** هو منتج SaaS متكامل يجمع بين خبرة المحاماة الرصينة وقوة الذكاء الاصطناعي السيادي.

## 🚀 ابدأ الرحلة
يمكنك الدخول إلى المنصة واكتشاف عالم من الاستشارات القانونية الذكية:
- **البوت الذكي**: استشارات فورية بأسلوب ChatGPT.
- **المكتبة السيادية**: إصدار عقود ونماذج معتمدة.
- **غرفة الخبراء**: تواصل مباشر مع نخبة المستشارين.

## 🛠 التقنيات المستخدمة
- **Next.js 15 & React 19**
- **Firebase (Firestore & Auth)**
- **Google Gemini AI (Genkit)**
- **Tailwind CSS & Framer Motion**
