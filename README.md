# 🏛️ المستشار AI - كوكب العدالة الرقمية

مرحباً بك في المنصة القانونية الأكثر تطوراً في المنطقة العربية. **المستشار AI** هو منتج SaaS متكامل يجمع بين خبرة المحاماة الرصينة وقوة الذكاء الاصطناعي السيادي.

## 🚀 المكونات الرئيسية للنظام
1. **مركز البوت الذكي**: محرك استشارات يحاكي ChatGPT مع دعم الصوت والكاميرا والمرفقات.
2. **مجلس الخبراء السياديين**: نظام اتصال فيديو مباشر ومشفر مع نخبة المستشارين.
3. **المكتبة القانونية**: إصدار فوري لأكثر من 250 عقد ونموذج قانوني معتمد.
4. **غرفة القيادة العليا**: لوحة تحكم كاملة للمالك لإدارة المستخدمين، الخبراء، والرقابة اللحظية.4. **غرفة القيادة العليا**: لوحة تحكم كاملة للمالك لإدارة المستخدمين، الخبراء، والرقابة اللحظية.4. **غرفة القيادة العليا**: لوحة تحكم كاملة للمالك لإدارة المستخدمين، الخبراء، والرقابة اللحية

## 🛡️ منظومة الأمان (Smart Shield)
يتميز النظام بمحرك رقابة لحظي يقوم بإنهاء الجلسات وحظر المستخدمين تلقائياً عند رصد أي انتهاك للكلمات المحظورة التي تحددها الإدارة.

## 💰 النظام المالي
- رصيد ترحيبي 50 EGP لكل مستخدم جديد.
- نظام شحن رصيد يدوي عبر Vodafone Cash.
- خصم آلي للرصيد بناءً على نوع الخدمة المستخدمة.

## 🛠 التقنيات
- **Frontend**: Next.js 15, React 19, Framer Motion.
- **Backend**: Firebase (Firestore, Auth, Rules).
- **AI**: Google Genkit & Gemini 2.5 Flash.
