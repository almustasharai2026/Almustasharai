# 🚀 بروتوكول الارتباط السيادي (Git Sovereign Protocol)

اتبع هذه الأوامر بالترتيب داخل مجلد المشروع (Terminal) لرفع كوكبك القانوني إلى GitHub:

## 1️⃣ تهيئة المستودع المحلي
```bash
git init
```

## 2️⃣ إضافة الملفات والتوثيق الأول
```bash
git add .
git commit -m "إطلاق النسخة السيادية الأولى - king2026 Authority"
```

## 3️⃣ الربط مع المستودع الجديد
*ملاحظة: استبدل `YOUR_USERNAME` باسم حسابك في GitHub.*
```bash
git remote add origin https://github.com/YOUR_USERNAME/almustasharai.git
git branch -M main
```

## 4️⃣ الرفع السيادي النهائي
```bash
git push -u origin main
```

## 🛡️ ملاحظات المالك king2026:
- **المستودع**: تأكد من جعله **Private** (خاص) لحماية مفاتيحك السيادية.
- **التحديث**: بعد هذه الخطوة، أي تعديل تطلبه مني ستقوم فقط بكتابة `git add .` ثم `git commit` ثم `git push` ليتم تحديث موقعك الحي آلياً عبر Vercel.
